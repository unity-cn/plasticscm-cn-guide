    4，如何Push（提交一个Checkin）？

    提交一个Checkin（Commit）

    当项目准备提交的时候，保存所做的修改，它们会出现在Checkin item窗口中，在Pending changes 窗口添加一个描述，勾选提交的Items，点击Checkin changes 完成第一次的Checkin

    查看您的提交

    提交后您将可以在Changesets中可以看到您的提交历史
